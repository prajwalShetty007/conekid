# Cone Kid

Static web game built with Vite and Phaser.

## Local development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
npm run preview
```

## GitHub Pages

This repository is configured to deploy automatically to GitHub Pages from the `main` branch using GitHub Actions.

Expected site URL:

```text
https://prajwalshetty007.github.io/conekid/
```
